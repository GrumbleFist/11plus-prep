# 11+ Verbal Reasoning: Detailed Research

## All 21 GL Assessment Question Types

### Type A: Insert a Letter
Two pairs of incomplete words. One letter completes first word AND begins second word in each pair.
- Easy: `BU(?)AP` / `DE(?)OT` → **N** (BUN/NAP, DEN/NOT)
- Strategy: Try consonants systematically. Check ALL FOUR words.

### Type B: Find Two Odd Words Out
Five words — three share a connection, two don't belong.
- Easy: Pig, Eel, Cow, Horse, Goldfish → Eel + Goldfish (aquatic vs land)
- Hard: Sonnet, Limerick, Novel, Haiku, Biography → Novel + Biography (prose vs poetry)

### Type C: Alphabet/Letter Codes
Word + its code given. Work out the letter-shift rule. Apply to encode/decode another word.
- Easy: constant shift (+1 each letter)
- Hard: alternating/increasing shifts, mirror codes (A↔Z)

### Type D: Synonyms
Two groups of three words. Find one from each group closest in meaning.
- Hard: (Obscure, Benign, Candid) + (Forthright, Malignant, Renowned) → Candid + Forthright

### Type E: Hidden Word
Four-letter word hidden spanning two adjacent words in a sentence.
- "I saw the bir**D OVE**r the hedge" → DOVE
- Strategy: slide along boundaries checking 3+1, 2+2, 1+3 splits

### Type F: Missing Three-Letter Word
Word in CAPITALS has three consecutive letters missing that form a valid word.
- "The PRIES walked slowly" → PRI[EST]S → EST

### Type G: Calculating with Letters
Letters assigned number values. Compute and convert back.
- A=1...Z=26: A + C + E = 1+3+5 = 9 = I
- Must follow BODMAS

### Type H: Antonyms
Two groups of three words. Find most opposite pair.
- Hard: (Verbose, Prudent, Benevolent) + (Malevolent, Reckless, Taciturn) → Benevolent + Malevolent

### Type I: Complete the Calculation
Balance an equation with one missing number.
- (12 + ?) × 2 = 8 × 3 + 6 → ? = 3

### Type J: Move a Letter
Remove one letter from first word (leaving valid word), insert into second word (making new valid word).
- BLAND + SINK → BAND + SLINK (move L)

### Type K: Number Relationships
Three groups of three numbers. Centre derived from outers by consistent rule.
- 2[12]6, 5[20]4, 3[?]9 → multiply: 27
- Hard: squared + added

### Type L: Letter Sequences
Pattern of letters with one missing.
- A, C, E, G, I, K, ? → M (+2)
- Hard: A, B, D, G, K, ? → P (+1,+2,+3,+4,+5)

### Type M: Word Analogies
"X is to (a,b,c) as Y is to (d,e,f)"
- Shoe:foot as glove:hand
- Author:book as composer:symphony

### Type N: Word-Number Codes
Four words, three codes (not in order). Match codes to words using letter patterns.
- Look for repeated letters matching repeated numbers
- Build cipher letter by letter, process of elimination

### Type O: Complete Word Pairs
Three pairs where second word derived from first by consistent rule.
- (SHOUT,SHOT) (SOLDER,SOLE) (FLUTED,?) → positions 1,2,3,5 → FLUE

### Type P: Number Sequences
Number pattern with one missing.
- Easy: +2 each time
- Hard: Fibonacci-like, square numbers, second-order differences

### Type Q: Compound Words
Combine one word from group 1 + one from group 2 to form valid compound word.
- (butter, cheese, bread) + (cake, cup, board) → buttercup

### Type R: Create a Word from Surrounding Words
Word in brackets formed from letters of words either side. Same pattern applied to new pair.

### Type S: Connecting Word (Similar Meaning)
Find single word from options relating to BOTH given pairs (polysemy).
- (cover, screen) + (colour, hue) → **shade**
- (fasten, lock) + (lightning, thunder) → **bolt**

### Type U: Letter Analogies
"CG is to ED as BW is to ?" — find shift for each letter position.
- First letters may follow different rule from second letters

### Type Z: Reading Information / Logic Problems
Passage provides information. Multi-step reasoning to answer questions.
- Organise with tables/timelines. Use only stated facts.

## GL vs CEM Differences

| Feature | GL | CEM |
|---|---|---|
| VR Paper | Separate, standalone (~50 min, ~80 questions) | Integrated with English |
| Question types | 21 standardised types | Comprehension, cloze, shuffled sentences |
| Format | Multiple choice | Mixed, timed sections (can't go back) |
| Predictability | High — practise specific types | Low — format changes yearly |
| Key skill | Logic + pattern recognition | Vocabulary breadth |
| Preparation | Systematic type practice | Extensive wide reading |

## Vocabulary Requirements
- Single biggest differentiator in VR performance
- Need 300+ targeted words minimum, ideally 2,000+
- Cannot be crammed — requires months of reading
- Must understand: synonyms/antonyms, homonyms, polysemy, word families, analogies
- Latin/Greek roots essential for decoding unfamiliar words

## Common Mistakes
1. Misreading the question (synonym vs antonym)
2. Spelling errors in word manipulation
3. Not checking all conditions (finding letter for one pair, not both)
4. BODMAS failures in arithmetic types
5. Incomplete logic in reading information
6. Time mismanagement (3 min on one hard question, 5 easy ones unanswered)
7. Vocabulary gaps → random guessing
8. Alphabet miscounting (especially middle of alphabet)
9. Not attempting all questions (no penalty for wrong answers in GL)
10. Panic on unfamiliar formats
